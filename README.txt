BEHAVIORAL FIM - QUICK START GUIDE
================================

1. HOW TO RUN
   Double-click the "start_fim.bat" file.
   - It will open a black terminal window (The FIM Core).
   - It will automatically open your web browser to the Dashboard.

2. HOW TO TEST
   - Go to the "test_monitored" folder (created automatically).
   - Create a new file -> Watch the Dashboard log.
   - Modify a file -> Watch the Dashboard log.
   - Open "passwords.txt" (Canary File) -> HEAR the Alarm and see the Red Alert.

3. HOW TO STOP
   - Simply close the "FIM Core" terminal window.

4. PRESENTATION
   - Run "python create_ppt.py" to generate your slides.
   - Run "python create_report.py" to generate your project report.

Good luck with your project!
